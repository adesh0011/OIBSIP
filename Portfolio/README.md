# Portfolio-Website
This repository contains a static portfolio website created using HTML, CSS, and JavaScript.
The website is designed to showcase projects, skills, and achievements, but it is not responsive and is best viewed on desktop devices.

✨ Features

👤 About Section – Basic introduction and career overview

💻 Projects Section – Highlights of sample projects with details

🏆 Skills Section – Lists of technical skills and tools used

📬 Contact Section – Email/social media placeholders

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

🚀 Future Scope

📱 Add responsiveness for mobile & tablet screens

🎨 Improve UI/UX design

🔗 Add hosting support for live demo

⚡ This project is mainly for practice and learning purposes.
